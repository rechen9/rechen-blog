一、VsCode拉取代码到Github仓库

1.初始化git仓库
  git init

2.添加所有文件到暂存区
  git add .

3.提交更改
  git commit -m "Your commit message"

4.添加GitHub仓库
  git remote add origin "https://github.com/rechen9/mypersonal-blog.git"

5.推送到Github
  git push -u origin master
